# Wattson

Wattson is a private SwiftUI iPhone and iPad app for the Glassow household.

It keeps the day visible with a shared schedule, chores, bulletin notes, and a meal plan. The first version stores household data locally on-device.

## Open

Open `Wattson.xcodeproj` in Xcode.

## Signing

The project is configured for automatic signing with this bundle identifier:

```text
com.glassow.wattson
```

In Xcode, select the Wattson target, choose your Apple Developer team under **Signing & Capabilities**, then build for iPhone or iPad.
